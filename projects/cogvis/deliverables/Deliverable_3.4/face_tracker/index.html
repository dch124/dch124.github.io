Face Tracker Software

This is a reimplementation of the Manchester face tracker, extended to work with spline curves rather than landmark points and also to work in color.

The tracker works by matching a face model to a movie input (or live camera). The face model can generate a face with different expressions by varying parameters. 4 affines (x,y,scale,rotation) and an defined number of combined ones (shape + color eigen components). The tracker updates the face model via an error model previously learned.

The code available from this page was used to provide results for the paper:

Reactive Memories : An Interactive Talking Head
V. E. Devin and D. C. Hogg,
Published in The British Machine Vision Conference 2001
